#### 贡献

方式：创建PR。

首先，你要在index2.html文件里添加你的栏目
添加的代码如下

```
<div class="layui-col-xs12 layui-col-sm12 layui-col-md6 layui-col-lg6 ">
              <a href="./snippets/你的html文件名称"><button class="layui-btn-fluid"><h1>类别</h1><br><h3>你的片段名称</h3></button>
              </a>
                  </div>
```


 添加在"请在这里贡献代码"的上一行，上面离你最近的</div>的下面，添加这个代码，不要删除下面的"请在这里贡献代码"。
 然后在snippets文件后面添加你的html文件，如果你的代码所在类别，这里已包括，可以在demo文件夹里寻找模板。上面的代码中href后面就是你的html文件名称和所在位置，snippets
一般来说你只需要改"你的html文件名称"即可。如果你想自定义也欢迎，可以不用模板，可以把文件地址改为你的链接，但是，要注意，你的贡献
必须要有3个栏目，预览，源码，详解，如果缺少，则不会通过 你的pr,另外，广告不能影响使用体验。
暂时就这些，随着项目的发展在完善吧。
